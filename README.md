## 📱 Sobre o Projeto

Este projeto foi criado para oferecer uma experiência moderna e envolvente, simulando a abertura de um convite físico de forma digital.

Ao acessar a aplicação, o usuário interage com a interface como se estivesse abrindo um envelope real, tornando a experiência mais memorável e intuitiva.

### Principais recursos

* Abrir o convite de forma interativa;
* Confirmar presença diretamente pelo WhatsApp;
* Visualizar sugestões de presente;
* Acessar a localização do evento via Google Maps;
* Navegar entre telas com transições fluidas;
* Ouvir música de fundo durante a experiência.

---

## 🎯 Funcionalidades

✅ Abertura interativa do convite

✅ Navegação intuitiva entre telas

✅ Botões posicionados estrategicamente sobre imagens

✅ Integração com WhatsApp

✅ Integração com Google Maps

✅ Sugestão de presente com opção de pagamento via PIX

✅ Música de fundo durante a navegação

✅ Layout responsivo para dispositivos móveis

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* LocalStorage

## 🧠 Aprendizados

Durante o desenvolvimento deste projeto, foram aplicados e aprimorados conhecimentos em:

* Manipulação do DOM;
* Controle de navegação entre telas;
* Persistência de dados utilizando LocalStorage;
